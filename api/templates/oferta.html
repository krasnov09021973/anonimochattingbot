2.1. Исполнитель предоставляет Пользователю возможность использования функционала Бота для анонимного общения с другими пользователями и/или AI-собеседником.

2.2. Конфиденциальность и хранение данных:

  • Бот не сохраняет историю переписки в своей базе данных.

3. Функционал Бота

3.1. Бесплатный доступ:

  • Поиск и общение с живыми собеседниками (до лимита в день).
  • Общение с AI-собеседником (до 50 сообщений).
  • Редактирование профиля (имя, возраст, пол, фото).
  • Выбор тем для поиска.

3.2. Премиум-доступ (платный):

  • Безлимитное количество чатов в день.
  • Приоритетный поиск собеседника (более быстрое соединение).
  • Общение с AI-собеседником без ограничений.
  • Приоритетная поддержка.
